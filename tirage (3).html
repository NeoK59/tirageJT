Guide d'utilisation

Application de tirage au sort pour les jeux de tables du casino.
Deux fenêtres : cette page pour l'opérateur, l'écran public projeté sur grand écran.
1
Préparer les tables
Dans la colonne de gauche, cochez les tables qui participent au tirage. Cliquez sur un siège pour l'exclure individuellement (siège vide, réservé…). Le résumé en haut à droite affiche le nombre de participants actifs.
2
Ouvrir l'écran public
Cliquez sur Écran public en haut à droite. Une nouvelle fenêtre s'ouvre — projetez-la sur le grand écran. Elle affiche l'écran d'attente animé jusqu'au prochain tirage.
3
Effectuer le tirage
Cliquez sur Effectuer le tirage. Les mini-rouleaux tournent et s'arrêtent sur la table et le joueur gagnant. Vérifiez le résultat affiché.
4
Lancer l'animation
Cliquez sur Valider & Lancer l'animation. L'écran public se lance : les rouleaux défilent puis s'arrêtent sur le gagnant avec confettis. Le résultat est ajouté à l'historique.
5
Terminer le tirage
Après remise du prix, cliquez sur Tirage terminé pour ramener l'écran public à l'écran d'attente. Vous pouvez relancer un tirage précédent depuis l'historique via le bouton ↺.
⚙
Personnaliser l'affichage
Le bouton Réglages permet de modifier les textes, la police, le thème de couleurs, le fond d'écran, le logo, et les dimensions des rouleaux. Un aperçu en temps réel est visible à droite. Tout changement est transmis à l'écran public après enregistrement.
♠
Gérer les tables et sièges
Dans Réglages → onglet Tables & Sièges, vous pouvez renommer les tables, ajouter ou supprimer des sièges, les réordonner par glisser-déposer, et ajouter de nouvelles tables.
♦   Les paramètres et l'historique sont sauvegardés automatiquement dans le navigateur.   ♦
